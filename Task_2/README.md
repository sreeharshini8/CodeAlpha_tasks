# 📊 Unemployment Analysis with Python

An end-to-end data analytics project focused on understanding unemployment trends, identifying regional patterns, and analyzing the impact of Covid-19 using Python.

---

# 🎯 Project Objective

The main objectives of this project are:

- Analyze unemployment rate data across different regions.
- Clean and preprocess raw unemployment datasets.
- Explore unemployment trends over time.
- Study the impact of Covid-19 on unemployment levels.
- Identify seasonal and regional patterns.
- Generate actionable insights for economic and social policies.

---

# 🛠 Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

# 📂 Dataset Information

The dataset contains:

- Date
- Region / State
- Estimated Unemployment Rate (%)
- Labor Participation Rate
- Employment Estimates

---

# 🔍 Project Workflow

## 1. Data Cleaning

- Checked missing values.
- Removed duplicate records.
- Standardized column names.
- Converted date formats.

## 2. Exploratory Data Analysis

- Analyzed unemployment distribution.
- Compared region-wise unemployment.
- Studied monthly and yearly trends.

## 3. Data Visualization

Created:

- Line Charts
- Histograms
- Boxplots
- Heatmaps

## 4. Covid-19 Impact Analysis

- Compared unemployment before pandemic.
- Compared unemployment during pandemic.
- Analyzed recovery trends.

---

# 📈 Key Insights

- Unemployment increased sharply during Covid-19.
- Some regions were affected more than others.
- Seasonal trends were observed in employment data.
- Recovery patterns varied by region.

---

# 💡 Policy Recommendations

- Focus employment support on high-risk regions.
- Strengthen skill development programs.
- Monitor labor market trends continuously.
