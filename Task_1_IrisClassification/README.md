# 🌸 Iris Flower Classification using Machine Learning

## 📌 Project Overview

This project focuses on classifying Iris flower species using Machine Learning and Python.  
The model predicts the species of an Iris flower based on flower measurements.

This project demonstrates the complete machine learning workflow including data loading, preprocessing, visualization, model training, prediction, and performance evaluation.

---

## 🎯 Objective

- Classify Iris flowers into different species  
- Build a machine learning classification model  
- Analyze flower measurement data  
- Evaluate model performance  
- Understand supervised learning concepts  

---

## 🌺 Dataset Information

This project uses the Iris dataset from Scikit-learn.

The dataset contains 150 flower samples from three species:

- Iris Setosa  
- Iris Versicolor  
- Iris Virginica  

Each flower contains four input features:

- Sepal Length  
- Sepal Width  
- Petal Length  
- Petal Width
